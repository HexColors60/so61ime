取碼方式

在本質上，六一輸入法，任何一個字皆取三碼，碼統一長度將使取碼更為方便與精準。如果碼數不足，則以『/』補之。六一輸入法定義的三個碼為『首碼、尾碼、補碼』。其中補碼是比較難取的碼，而首碼、尾碼是相對而言比較簡單的，所以，六一輸入法特別採用尾碼提前取的方式來進行拆碼，

對於字體本身如果可以分割的話，字首取一碼，字身取兩碼，兩碼分別是字身的首碼及尾碼，如果字身只有一碼，而字首有兩碼或以上，則補碼取字首的尾碼，相反的如果不能分割，則全部取三碼。

字體的分割方式：

一、字首以分出最小單位為原則

翁：字首的部分只有 ，而不是公，字首的決定不是在一般所謂的部首，而是以字體為準，所能分割的最小單位。

麗：這個字型，字首為『一一』，故首碼應該取『一』，尾碼為『匕』取二，此字的補碼應取字身的首碼，為取３。
當：字首的部分只有口上方的字，而不是尚。
受：斜月的字，下面的字視為相連，故此字身為又，其餘的字即為字首。
愛：斜月的字，下面的字視為相連，故此字身為心，其餘的字即為字首。
二、分內外的字，其外部的字需獨立，不可混雜。

囟：這個字，其外面的碼，有混雜其他字型，所以整個字體視為不可分割。

面：這個字，其外面的碼，有混雜其他字型，所以整個字體視為不可分割。

圓：這個字，為可分割字型，外圍的字元先取為『囗』，再來取裡面的尾碼，為貝取３，再取補碼口取０。

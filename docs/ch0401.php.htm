六個代碼

代碼是六一輸入法很奇特的方式之一。有些碼符合的字太多，其中又以『首碼』居多，如木、人、這樣的字，如果沒有想出一個辦法來處理，三碼(30鍵)要決定一個字會有相當的困難，為了要讓三碼可以決定一個字，所以，我們採用了代碼的方式來處理。

所有的輸入法都是用字根是什麼碼，就取什麼碼的方式來取碼，但是在六一輸入法裡，有一些『例外』，這些例外就是現在要跟大家說明的替代碼方式。

六個代碼是六一輸入法裡取碼中很重要的一部分，請您務必要學會，六一輸入法的本質是只要首尾碼會拆就可以了，而這一節的六個代碼是針對首碼而來，如果沒弄懂，很難學會六一輸入法。

注意：六一輸入法裡的代碼只用在『首碼』，代碼都是在原碼的上方，且大部分在正上方。

二為卄的代碼
當某一個字的首碼原本應是廾時，首碼有兩種可能，
１、『取廾』，也有一種可能，就是
２、『取二』，您必須分辨在什麼條件下應該要取『廾』，什麼條件下，要改取『二』。

當一個字的首碼原本應取廾時，如符合下列情況者，改取代碼二
一、首碼不為廾字頭在正上方（首碼為其他輔助字根時），如：
前頭觀型首副益歡豆

二、字體（去掉廾字頭之後）可再分左右兩邊，如：
范苑茆茫荊荏茹茨莎莊莉荷荻菸萍菠菰萌菽菲菔落葫葩葭
葆蒞蒲蒜蓀蓓蔽蔚蔬蔭蔣蔆蕩

三、且非一碼字
丱，此字雖然不是廾字頭，但因其為一碼字，故取原碼。

換個角度來看好了，如果字體是最常見的『廾字頭在最上方』，（才有可能）取廾，否則取代碼『二』，以下的字取廾（廾字頭且不能分左右兩邊）

３為三的代碼

首碼三的字有兩種取碼方式，『三』或『３』。
中國字的三點水真的很多很多，所以我們才想出這個辦法來處理這個問題，有很多很多字都取到『３』這一碼來了，像『沒汽淡沉』這幾個字都取到『３』這一碼來。
如果字體的首碼不為或字體可再分上下兩個，且非一碼字，則取代碼３。
字體分上下兩個，其實是比較難分的，有時會有相當的盲點，可能得花一些時間來適應。
如：沒，原應為三，因剩下的字可再分上下兩個，故取代碼３，尾碼為『又』取『三』，常用字補碼略，故為『３三』。
例外：如果字體只有分開廾字頭或皿字底，不算。
如：漭，雖然可以分上下兩個，但只分了廾字頭，故不取替代碼，首碼取三，尾碼取廾，補碼亦為廾，得為『三廾廾』。
溢這個字雖然可以分上下兩個，但只分了皿字底，故不取替代碼，首碼取三，尾碼取廾，補碼亦為廾，得為『三廾廾』。
最終定義如下：
定義 當一個字首碼原為『三』時，如果字體的首碼不為或(字體可再分上下兩個且不是只分出廾或皿)，且非一碼字，則取代碼３。

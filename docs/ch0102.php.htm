六一輸入法0.8版，系統特色

輸入法程式

六一輸入法的本質是只取首尾兩碼，任何一個字，只要會取首尾兩碼則都可以順利打出字來，有了六一輸入法，您再也不用擔心有打不出字的困擾。

人類的大腦有限，沒有必要去記太多的東西，六一輸入法『全面』而『完整』的前導式提示，讓您打字完全不需要額外的記憶，如果有不會打的字，只要稍微看一下目前的前導式字碼表，即可輕鬆打出該字，這是六一輸入法的特色，也是六一輸入法設計的主軸。

全面而完整的提示

不知大家會不會玩橋牌，當叫牌結束時，有一家的牌是要翻開給大家看的，然而，台灣有些人玩橋牌，卻是四個人矇著頭在打的，現今的輸入法就像後者，而六一輸入法則是前者。

有了這些前導式提示，三碼定江山，才成為可能，在六一輸入法裡，理論上，所有的字都和兩碼字一樣快。

視窗可自行調整大小

倘若使用者覺得系統的視窗太小，六一輸入法允許您調整字型大一點，另外，如果您夠熟了，可以將動態鍵盤功能先行關閉，待必要時再予以開啟。

大視窗：『∕尸１』
小字體：『∕尸一』
隱藏提示窗格：『∕尸竹』

在隱藏視窗的模式下，當輸入兩個按鍵之後，只要打入『'』程式即會列示提示視窗一次，讓使用者可以鍵入突然忘記或是不太常用的字體，輸入完，程式自動再關閉提示視窗。

單鍵出字

這是六一輸入法特別設計的功能之一，有一些字有時候會在同一時間點不斷輸出，我們將這些有高度相關性的字，放在一起，方便使用者使用。

例如：『新台幣伍仟肆佰玖拾捌萬零壹佰肆拾捌元整』 這樣的字在六一輸入法裡只要打『3＋19+1』個鍵，即可輸出這十九個字。 首先鍵入『∕∕』進入單鍵模式，再按『七』進入大寫的數字模式，上面的字即可單鍵出字，最後打上『空白鍵』結束單鍵輸入模式即可。

符號專家

不管符號表設計的再精良，皆不若用選的來的快，來的好，六一輸入法的全面而完整的提示功能，對於符號的輸入更為方便。

首先打『∕∕』進入單鍵出字模式，之後再打入『２３４５６』等任一鍵，即可進入單鍵符號模式，之後您可以在此符號表下『單鍵』出字『㊣◎○●⊕⊙△▲☆★◇◆□■▽▼』，即可以極快的速度打出您所想要撰打的符號。

半形輸入也可以

由於在打字的時候，還是有可能需要打出半形字，目前市占率較大的幾個輸入法，似乎不認為半形輸入是輸入法該做的事，六一輸入法特別讓使用者，可以在某些情形下，不用任何切換即可輸出半形字。如『,』這個半形字，字碼為『∕竹』，另外在單鍵出字的模式下，如『∕∕乂』、『∕∕手』、『∕∕木』、『∕∕人』、『∕∕尸』、『∕∕∕』就可以在六一輸入法下打出半形字，不需要先關掉輸入法，打完再切回來。

第二節、鍵位說明

為了能議使用者更方便記住鍵盤的位置，我們特別使用了數字碼當做字根。
六一輸入法，使用三十個鍵來操作，如下所示

另外，我們還保留了一個鍵，單引號『'』
『'』的特殊用法，請參考V0.8簡介。

１２３４５６７８９０
一二三四五六七八九十
竹卄廿火乂手木人尸∕
練習：字根形之篇
因為上排和中排的讀音都一樣，所以，我們特別又弄了一個以字意為發音的列表，如此方能以讀音分辦所屬正確的字，
中乙貝心彎女刀日月口
一二三四五六土八九十
竹草散火叉手木人耳空

練習：字根義之篇 另外，如果可以接受兩個音節的話，可以加上『上排、中排、下排』，

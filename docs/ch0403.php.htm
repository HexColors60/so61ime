８為八的代碼

首碼八的字有兩種取碼方式，『八』或『８』。
如果字體的首碼不為『金』字，且非一碼字，則取代碼８。
如：以，原應為八，因其字不為『金』這個字型，故取代碼為８，尾碼取人，補碼為『、』取二，故為『８人二』。
如：處，首碼原麼為八，因其字不為『金』這個字型，故取代碼為８，尾碼為『几』取『九』，補碼為『』取『乂』，常用字，補碼略，故為『８九』。
秘訣當一個字首碼原為『八』時，如果字體的首碼不為金字，且非一碼字，則取代碼８

９為人的代碼

首碼為『人』的字有兩種取碼方式，『人』或『９』。
如果字體的首碼不為?字邊，且非一碼字，則取代碼９。
但，『隹』這個字型也取代碼９，這個字本身取『９手∕』。
集，首碼隹取９，尾碼取木，補碼 取手，全字取為『９木手』
您，這個字首碼取人，不是代碼，首碼是就取人，同樣的，字首是金就取八，首碼是就取手，這三種字型還不算太多，沒有再和『廾木』一樣繼續往下分了。
在鍵盤上，人和９跳一格，又歪一邊了。

秘訣當一個字首碼原為『人』時，如果字體的首碼不為字邊，且非一碼字，則取代碼９。

最終回顧：
一、一碼字取原本的碼
二、不含『卄木金』字型的字，必然取代碼。
三、廾，木如果可以再分左右兩邊，取代碼。
四、如果可以再分上下兩個，取代碼。
五、『卄木』如果不是在最常用的位子，取代碼；『金』只要字型是原本的字型則不取代碼。

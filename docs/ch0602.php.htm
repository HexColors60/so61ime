取碼規定

統一取碼

統一取碼會造成更多的衝碼，如果不統一，可能造成取碼的混淆甚至錯誤，兩相權害取其輕，為了讓使用者取碼更便利，六一輸入法採用不會造成混淆的方式來處理，那就是我們現在要介紹的統一取碼原則。

采釆
采釆這兩個字上面的部分，都極為相似，所以我們將之取為一樣的碼，斜月取９。這樣的取碼方式，可避免一些取錯碼的困擾。如設將『釆』取為『竹木火』那麼其他相關的字型，將會受到嚴重的影響

戈七匕弋
這四種字為求統一，這四個字型一律取二，『戈』和『、』就倉頡來說，是取同一碼。

恆恒
恆恒這兩個字差不多是一樣的，所以，『、日』我們特別將之統一取碼為８。

上下都是十

朝車，下方的十一概不取
車這個字時常用到，如果是取尾碼的時候，因為下面的『十』一律不取，所以記得是要取『四』，而不是『十』。
如連這個字，首碼為『』取『乂』，尾碼為『田』取『四』，補碼取十，整個字碼為『乂四十』。
如運這個字，首碼為『』取『乂』，尾碼為『田』取『四』，補碼為取３，整個字碼為『乂四３』。
如朝這個字，首碼取『十』，尾碼為『月』取『９』，補碼為『日』取『８』，整個字碼為『十９８』。

行、門、鬥

行、門、鬥字算字首，其餘根據字首之取碼法則處理(字首取一碼，字身取兩碼)。尢其行這個字要特別小心。
衝：這個字，首碼為彳取尸，尾碼取七（注意行為字首，所以不是取丁），補碼取十（為捨碼），整個字得碼為『尸七十』。
鬧：這個字，首碼為『』取１，尾碼需取裡面的巾，而不是外面的，補碼為〦，此字字身尚有〦，依據規則，字身應取兩碼，故本字碼為『１五６』。
問：首碼取日，尾碼則取口，字身一碼已取完，補碼取字首的末碼，此時應取?，故本字碼為『８０７』。
